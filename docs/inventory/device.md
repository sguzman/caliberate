# Device Inventory (Calibre Reference)

## Core device modules
- `calibre/src/calibre/devices/manager.py` (device manager)
- `calibre/src/calibre/devices/usbms/driver.py` (USB mass storage driver)
- `calibre/src/calibre/devices/interface.py` (device interface contracts)

Parity targets
- Detect connected USB mass storage devices.
- Enumerate device libraries and supported formats.
- Send-to-device and cleanup operations.
